## Startup Instructions

At the beginning of every session, you MUST read and follow these files to understand the project structure, security policy, and contributing guidelines:
- [AGENTS.md](file:///home/thruqe/whatsrook/AGENTS.md) - Deep codebase architecture, layout, and how to write new commands.
- [CODE_OF_CONDUCT.md](file:///home/thruqe/whatsrook/CODE_OF_CONDUCT.md) - Expected standard of behavior.
- [SECURITY.md](file:///home/thruqe/whatsrook/SECURITY.md) - Crucial security policies regarding credentials, memory safety, and acceptable use.

## Command Reference

- Format: `go fmt ./...`
- Vet: `go vet ./...`
- Lint: `golangci-lint run`
- Test: `go test ./...`
- Build: `go build -o whatsrook .`
